Free PS5 & PS4 Controller Diagnostic Tool

Chrome or Edge Required

Controller Diagnostic Lab uses the WebHID API to communicate directly with your PS4/PS5 controller over USB. This API is only available in Chrome and Edge — it's a browser platform limitation, not something we can work around.

Download Chrome Download Edge
Why WebHID? Standard web APIs can only read basic gamepad buttons. WebHID gives us raw HID access so we can read calibration data, trigger haptics, adjust dead zones, and write directly to the controller's memory — features impossible in any other browser.

Laptop Battery Health Tester

Diagnose your battery capacity, cycle count, and see estimated replacement options.

Run Battery Diagnostics

Web browsers restrict direct access to hardware cycle counts. Please use the instructions below to get your data, and we will analyze your health and replacement costs.

Windows Users

1. Right-click Start, open Terminal/PowerShell

2. Type: powercfg /batteryreport

3. Open the generated HTML file in your folder to find Design Capacity and Full Charge Capacity.

Mac Users

1. Hold Option (⌥) and click the Apple menu 

2. Select System Information

3. Click Power on the left side to find your Cycle Count and Full Charge Capacity.
